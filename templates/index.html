Welcome to PittCTF, an online Capture the Flag platform.


PittCTF is a website where anyone can come and attempt to complete security challenges that test skills in forensics, web exploitation, scripting, binary reversing, and many other skills. Users can add their own challenges and also make competitive lobbies both private and public for other users to compete in!

How to use the platform:


The Public Lobby is an area where users have the ability to attempt the challenges uploaded by the PittCTF Team.

The Public Leaderboard only reflects the challenges created by the PittCTF Team.

Within Challenges, you have the ability to create challenges and edit the challenges you have already created.

Once you have created challenges, you can create your own lobby to host private challenges with a leaderboard unique to the lobby.
